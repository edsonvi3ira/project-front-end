No CSS, usamos a propriedade para especificar a fonte de um texto : font-family

A propriedade é usada principalmente para especificar texto em itálico : font-style Esta propriedade tem três valores:

  • normal - O texto é mostrado normalmente

  • itálico - O texto é mostrado em itálico

  • oblíquo - O texto é "inclinado"
    (oblíquo é muito semelhante ao itálico, mas menos suportado)

A propriedade define o tamanho do texto : font-size Ser capaz de gerenciar o tamanho do texto é importante no web design.
No entanto, você não deve usar ajustes de tamanho de fonte para fazer com que os parágrafos pareçam títulos,
ou os títulos se parecem com parágrafos.
font-size: 40px;
font-size: 40rem;

A propriedade CSS Font

Para encurtar o código, também é possível especificar todas as propriedades de fonte individuais em uma propriedade.
A fontpropriedade é uma propriedade abreviada para:
  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family
Nota:: Os valores e são obrigatórios. Se um dos outros valores estiver ausente, seu valor padrão será usado font-size font-family
